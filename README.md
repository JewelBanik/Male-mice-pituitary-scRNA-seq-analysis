# Male-mice-pituitary-scRNA-seq-analysis

https://doi.org/10.1210/en.2018-00750

Reanalysis of this publicly available male mice pituitary scRNA seq dataset for Musashi1 and Musashi2. The findings of this dataset are comparable to https://doi.org/10.1210/endocr/bqaa245 and Ho et. al. dataset. (https://doi.org/10.1007/s13238-020-00705-x)
